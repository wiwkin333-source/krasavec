Favicon pack
============

Generated from: ChatGPT Image 18 июл. 2026 г., 18_59_18.png
Contents cover: modern browsers, iOS Safari, Android Chrome, Windows tiles, Safari pinned tabs.

INSTALL
-------
1. Unzip and upload ALL files to the root of your site (so they live at https://example.com/favicon.ico etc.).
2. Put this in the <head> of every HTML page:

    <link rel="icon" type="image/x-icon" href="/favicon.ico">
    <link rel="icon" type="image/png" sizes="16x16"  href="/favicon-16x16.png">
    <link rel="icon" type="image/png" sizes="32x32"  href="/favicon-32x32.png">
    <link rel="icon" type="image/png" sizes="96x96"  href="/favicon-96x96.png">
    <link rel="apple-touch-icon"        sizes="180x180" href="/apple-touch-icon.png">
    <link rel="apple-touch-icon-precomposed" sizes="180x180" href="/apple-touch-icon-precomposed.png">
    <link rel="mask-icon" href="/safari-pinned-tab.svg" color="#000000">
    <link rel="manifest" href="/site.webmanifest">
    <meta name="msapplication-TileColor" content="#ffffff">
    <meta name="msapplication-config" content="/browserconfig.xml">
    <meta name="theme-color" content="#ffffff">

3. (Optional) Edit site.webmanifest: set "name" / "short_name" / "theme_color" / "background_color"
   to match your brand.
4. (Optional) Edit the <meta name="theme-color"> and mask-icon color to match your brand.
